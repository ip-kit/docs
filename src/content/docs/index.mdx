## Quick Install

Add IPKit to Claude Desktop in one step:

```json
// ~/Library/Application Support/Claude/claude_desktop_config.json
{
  "mcpServers": {
    "ipkit": {
      "command": "npx",
      "args": [
        "-y", "mcp-remote",
        "https://ipkit.fly.dev/mcp",
        "--header", "Authorization:Bearer YOUR_API_KEY"
      ]
    }
  }
}
```

Then restart Claude Desktop and ask: *"Search for trademarks similar to 'CloudBooks' in the US and EU."*
